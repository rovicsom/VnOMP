Malicious Programs Security Report

QA & Security Course Overview

Introduction

Malicious programs, or malware, are designed to infiltrate, damage, or gain unauthorized access to computer systems. This report covers the major types of malware, their characteristics, and how they spread.

Key Takeaway:

Understanding malware types helps in implementing appropriate security measures and response strategies.

Types of Malicious Programs

Viruses

  • Self-replicating code that attaches to legitimate programs
  • Spreads when infected files are shared or executed
  • Can corrupt, delete, or steal data
  • Think of it like: A parasite that needs a host - attaches to your Word document and spreads when you email it to colleagues

Worms

  • Self-propagating malware that spreads across networks
  • Does not require host files to replicate
  • Consumes bandwidth and system resources
  • Think of it like: A self-driving car that copies itself - moves independently through your network without needing anyone to carry it

Trojans

  • Disguised as legitimate software
  • Creates backdoors for unauthorized access
  • Often used to steal credentials or install other malware
  • Think of it like: A fake delivery package - looks like legitimate software (free game, PDF reader) but contains hidden malware

Ransomware

  • Encrypts user files and demands payment
  • Major threat to businesses and individuals
  • Can spread through email attachments or network vulnerabilities
  • Think of it like: A digital kidnapper - locks up your files and demands ransom money to give you the key

Spyware

  • Secretly monitors user activity
  • Collects personal information, keystrokes, browsing habits
  • Often bundled with legitimate software
  • Think of it like: A hidden camera in your office - secretly watches everything you do and reports back to criminals

Rootkits

  • Hide deep within operating system
  • Difficult to detect and remove
  • Provide persistent unauthorized access
  • Think of it like: A master key that also makes itself invisible - gives hackers admin access while hiding from security scans

Common Attack Vectors

Email attachments

Malicious files disguised as documents

Example: "Invoice.pdf.exe" that looks like a PDF but is actually malware

Downloads

Infected software from untrusted sources

Example: Free movie download site that bundles spyware with the video file

USB devices

Removable media carrying malware

Example: USB stick found in parking lot contains malware that runs when plugged in

Network exploits

Taking advantage of system vulnerabilities

Example: Unpatched Windows systems infected by WannaCry ransomware in 2017

Social engineering

Tricking users into installing malware

Example: Fake tech support call claiming "your computer is infected, download our scanner"

Key Differences Summary

Speed of Spread

  • Fastest: Worms (automatic network spread)
  • Slowest: Viruses (requires human action)

Stealth Level

  • Most Hidden: Rootkits, Spyware
  • Most Obvious: Ransomware (announces itself)

Damage Type

  • Immediate Financial: Ransomware
  • Long-term Privacy: Spyware
  • System Destruction: Viruses, Worms

User Involvement

  • Requires User Action: Viruses, Trojans
  • No User Action Needed: Worms, Rootkits
